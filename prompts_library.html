Guide d'utilisation des prompts

Les prompts sont des instructions optimisées pour les modèles d'intelligence artificielle comme ChatGPT et Google Gemini. Ils sont conçus pour vous aider à accompagner efficacement vos coachés à travers les 10 étapes de votre méthodologie.

Chaque prompt peut être utilisé de trois façons :

  • Copier : Pour coller manuellement dans l'interface de votre choix
  • Ouvrir dans ChatGPT : Pour démarrer directement une conversation avec le prompt préchargé
  • Ouvrir dans Gemini : Pour utiliser le prompt avec Google Gemini

Vous pouvez également modifier les prompts selon vos besoins spécifiques.
